Summary: Latitude vs X

This project performs a weather analysis across 500 randomly selected cities. Weather data were requested using Open Weather Map API. The initial hypothesis was that there is a correlation between temperature, humidity, cloudiness and wind speed and latitude. The analysis shows that there is a strong correlation between latitude and temperature, and a weak correlation between latitude and humidity.
